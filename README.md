
    # Custom GPT Chatbot
    This is a simple custom chatbot using OpenAI's GPT model and Flask.
    
    ## How to run:
    1. Install dependencies:
       ```
       pip install flask openai
       ```
    2. Set up your OpenAI API key in `app.py`.
    3. Run the app:
       ```
       python app.py
       ```
    4. Access the chatbot at `http://localhost:5000/`.
